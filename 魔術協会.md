# 魔術協会・魔法機関 設定案（たたき台）

## 1. 魔法と国家の関係（前提）

この世界において、**「魔術レベル＝国力（軍事力）」**である。
そのため、各国は独自の魔法研究機関を持ち、最高機密として扱っている。

*   **国家魔術師**：国家公務員。その国の軍事機密（独自開発された攻撃魔法や通信暗号）を扱うため、国外への移動や情報の持ち出しは厳しく制限される。
*   **技術障壁**：A国で開発された魔法理論は、B国では解読不能な「暗号化された術式」として運用されることが多い。

## 2. 国際魔術協会（Magical Association）

**「対魔族」**を掲げて設立された、建前上の永世中立・国際調整機関。
本部は王都の中立区画（あるいは魔術都市）にある。

*   **設立の理念**：
    *   「人類共通の敵である魔族に対抗するため、基礎魔法技術と対魔族戦術を共有する」。
    *   あくまで「基礎」と「対魔族」に限定されており、対人・対国家魔法は管轄外とされる（が、実際は監視している）。
*   **実態と政治**：
    *   **「魔術の国連」**：各国から派遣された理事によって運営されるが、実態は**「技術スパイと腹の探り合いの場」**。
    *   「危険な禁術の監視」を名目に、他国の独自研究に干渉しようとする派閥と、それを拒む国家派閥で常に揉めている。
    *   優秀な野良魔術師や、国に属さない研究者を「協会認定魔術師」として囲い込むリクルート機関でもある。

## 3. アカデミア（魔術学院）

協会が運営する教育機関。
世界中の才能ある若者が集まるが、ここも政治の縮図である。

*   **表の顔**：最先端の魔術を学べる最高の学び舎。
*   **裏の顔**：
    *   各国が自国の優秀な人材を送り込みつつ、他国の優秀な人材を「青田買い（引き抜き）」する場。
    *   卒業生は「母国に戻るか」「協会に残るか」で踏み絵を迫られる。
    *   ここを追放された「倫理観の欠如した天才」が、ギルド『狂天』に流れるルートが確立されている。

## 4. 勢力図と対立構造

*   **国際協調派（協会派）**：
    *   「魔族の脅威は去っていない。国家の枠を超えて技術統合すべき」と主張。
    *   理想主義的だが、一部は「世界統一政府」のような野心を抱く危険思想も混ざる。
*   **国家主権派（軍部・貴族）**：
    *   「協会は技術を盗もうとしている」と敵視。
    *   自国の秘匿魔法が流出することを極端に恐れ、協会への協力は最低限に留める。
*   **ギルドとの関わり**：
    *   **『巌窟王』**：協会からの依頼（対魔族調査、遺跡発掘）を正規ルートで請け負う優良パートナー。
    *   **『狂天』**：協会が定めた「禁忌」に触れた研究者たちの掃き溜め。協会は常に監視・討伐対象としているが、裏では彼らの成果（非人道的な実験データ）を極秘に買い取っているという噂もある。
