# 【絶対入手禁止】闇市で出回る「違法アーティファクト」カタログ ～これを持っていたら即・死刑！？～

**「強力なアイテムが欲しい」。冒険者なら誰もが抱く願望だ。**
**だが、正規ルートでは手に入らない「禁忌の品」には、それ相応の「呪い」と「罪」が付いてくる。本誌は読者の皆様が誤って手を出さないよう、注意喚起としてそのリストを公開する。**

## 1. 隷属の首輪（カオス・チョーカー）
*   **闇市相場：** 金貨50枚～
*   **効果：** 対象の首に装着し、魔力を通すと、装着者は主人の命令に逆らえなくなる。それどころか、命令に従う瞬間に脳髄を焼くほどの強烈な快楽が与えられるため、進んで隷属を望むようになる。
*   **危険性：** 王国法で所持・売買共に死刑対象。さらに、安物は暴走して装着者の首を飛ばす事故が多発中。
*   **入手ルートとされる場所：** 貧民街の雑居ビルの古道具屋の合言葉「散歩の時間だ」

## 2. 偽りの英雄剣（カース・ブリンガー）
*   **闇市相場：** 金貨100枚～
*   **効果：** 誰でも剣聖並みの剣技が使えるようになる。
*   **危険性：** 斬った敵の生命力をエネルギー源とするが、攻撃が外れると「使用者」からエネルギーを徴収する理不尽仕様。一度の空振りで寿命が10年縮むため、極度のプレッシャーで自滅する者が後を絶たない。また、通常時でもエネルギーを必要とするため、使用者はしばしば辻斬り化する。
*   **入手ルートとされる場所：** 剣の墓と呼ばれる廃棄場ダンジョン。

## 3. 痛喰らいの虫の卵（ペイン・イーター・エッグ）
*   **闇市相場：** 金貨30枚
*   **効果：** 飲み込むか傷口に擦り込むことで体内に寄生し、孵化した幼虫が宿主のあらゆる痛みを食べて完全に無効化する。
*   **危険性：** ダメージそのものは消えない。「腕が切断されているのに気づかず戦い続け、出血多量で死亡」というケースが後を絶たない。また、ペイン・イーターは常に痛みを求めるため、使用者の中枢神経に作用して自傷行為に走らせるようになる。カプセル剤と偽り、初心者冒険者に売りつけられる代表格。

## 4. 魅惑の香水（サキュバス・バースト・ポーション）
*   **闇市相場：** 銀貨50枚（小瓶）
*   **効果：** 異性を強烈に惹きつけるフェロモンを放つ。
*   **危険性：** 効果が強すぎて、人間だけでなくゴブリンやオークまでも呼び寄せてしまう。「モテすぎて死ぬ」を地で行くアイテム。


## 5. 透視の魔眼（ピープ・アイ）
*   **闇市相場：** 金貨80枚
*   **効果：** 服や壁を透視できる。
*   **危険性：** 使用中、激しい頭痛と共に視力が低下し、最終的に眼球が爆発する。「深淵を覗くとき、深淵もまた…」の通り、見てはいけないもの（悪霊など）まで見えるようになり発狂する者も。自爆テロリストと相性が良い。

## 6. （偽）エリクサー
*   **闇市相場：** 金貨10枚
*   **効果：** どんな怪我も一瞬で治る（ように感じる）。
*   **危険性：** 強力な再生促進剤だが、細胞の制御が効かない欠陥品。傷口から腕が生えたり、目玉が増殖したりといった異形化を招く。「治ったが、人間ではなくなった」という悲劇を生む劇薬。わかってて使う人もいる。

## 7. 強欲の財布（グリード・ウォレット）
*   **闇市相場：** 金貨5枚
*   **効果：** 入れた金貨が翌日には倍になる。
*   **危険性：** 本当に増えるが、その代償として所持者の「不幸」や「凶悪な魔物との遭遇率」が激増する。街中だろうとドラゴンが降ってくるレベルの災厄に見舞われるため、逆にこれを利用してレベル上げをする命知らずもいる。

## 8. 身代わり人形（スケープ・ラブドール）
*   **闇市相場：** 金貨200枚
*   **効果：** 致命傷を負った際、一度だけ人形が身代わりになってくれる。
*   **危険性：** 身代わりになった人形は、使用者の「親しい人間」を殺しに行く凶悪な魔物となる。自分が助かった代償として、最愛の人物が目の前で惨殺される地獄を見る。


## 9. 盗聴の巻貝（ウィスパー・シェル）
*   **闇市相場：** 銀貨80枚
*   **効果：** 遠くの会話が聞こえるようになる。
*   **危険性：** 「聞きたくない声（悪口、陰口、浮気現場）」ばかり優先的に拾う性格の悪い呪物。さらに、一度使うと「幻聴」として定着し、貝を手放しても一生他人の悪意ある声が聞こえ続けるようになる。精神が崩壊するまで逃れられない。

## 10. ゾンビ作成の札（ネクロ・タグ）
*   **闇市相場：** 金貨150枚
*   **効果：** 死体に貼ると蘇る。
*   **危険性：** 貼られると9割はゾンビとなる。1割は完全に復活し、死者は蘇り、知性も完全に戻る。だが、その際も肉体の腐敗は絶対に止まらない。意識がはっきりしたまま、自分の体が腐り落ちていく激痛と恐怖を味わい続けることになる。かつてこの生ゾンビを作成し、売買する邪悪な組織が問題となった。

## 11. 飛行の靴（イカロス・ブーツ）
*   **闇市相場：** 金貨300枚
*   **効果：** 空を自由に飛べるようになる。
*   **危険性：** 高度を下げる機能がないため、大概は成層圏まで上昇して凍死するか、魔力切れで落下死する。まれに足だけ千切れて飛んでいくこともある。「着陸機能」が別売りの詐欺商品。暗殺道具として大ヒットした。

## 12. 無限の水筒（スワンプ・フラスコ）
*   **闇市相場：** 金貨40枚
*   **効果：** 水が無限に湧き出る。
*   **危険性：** この水は飲んでも乾きが癒えることはないし、身体に吸収されることもない。無限に飲むことが可能でもあり、「蜃気楼水」と呼ばれる。

## 13. ドッペルゲンガーの鏡（ミラー・イメージ）
*   **闇市相場：** 金貨500枚
*   **効果：** 自分と瓜二つの分身を作り出す。
*   **危険性：** 本体が起きているときは言うことを聞くが、寝ているときは本体を殺して成り代わろうとする。

## 14. 万能鍵（スケルトン・キー）
*   **闇市相場：** 金貨120枚
*   **効果：** どんな扉も開けられる。
*   **危険性：** ただし、押し上げたピンの数だけ必ず骨折する。

## 15. 幸運のサイコロ（デビル・ダイス）
*   **闇市相場：** 金貨10枚
*   **効果：** ギャンブルで必ず勝てる。
*   **危険性：** 勝った金額と同じ価値の「寿命」や大切なもの、すなわち「内臓機能」「地位」「名誉」「家族」などが取立人によって没収される。大勝ちした翌日に心不全で死ぬケース多数。

## 16. 翻訳の飴（バベル・ドロップ）
*   **闇市相場：** 銀貨30枚
*   **効果：** 舐めている間、あらゆる言語が話せる。それは人語に限定されず、神の言語も使用可能であるという。
*   **危険性：** 副作用で脳が溶け、元の母国語を忘れる。さらに効果が切れると、一切の言葉が話せず「鳴き声」しか出せなくなる。

## 17. 無限の干し肉（トロール・ジャーキー）
*   **闇市相場：** 銀貨50枚
*   **効果：** どれだけかじっても、瞬時に再生して元の大きさに戻る魔法の保存食。
*   **危険性：** 材料は再生力が強すぎるトロールの死肉。胃の中に到達すると宿主の血肉を吸い取りながら急速に自己増殖し、最終的に宿主の腹を突き破って肉塊や子トロールが誕生する。食費を浮かせたい貧乏冒険者がよく犠牲になる。

## 18. 怪力の腕輪（オーガ・バングル）
*   **闇市相場：** 金貨60枚
*   **効果：** 大岩も砕く怪力が手に入る。
*   **危険性：** 恐るべき筋肉痛で脳が焼き切れる。しばしば痛喰らいの虫の卵（ペイン・イーター・エッグ）と併用される。

## 19. 忘却の砂（オブリビオン・サンド）
*   **闇市相場：** 銀貨40枚
*   **効果：** 嫌な記憶を消せる魔法の砂。
*   **危険性：** たいへんな依存性がある。

## 20. 惚れ薬・改（ラブ・ポーション・No.99）
*   **闇市相場：** 金貨80枚
*   **効果：** 飲ませた相手を強制的に惚れさせるが、効果が切れるとキスがスイッチとなって爆発する。
*   **危険性：** 爆発は物理。威力は半径50mを消し飛ばすレベル。愛する人に吹き飛ばされたくなければ、あるいは愛する人を木っ端微塵にしないためには、死ぬまでこの高価な薬を買い続けなければならない。「愛」という名のサブスクリプション地獄。

## 21. 影踏みの短剣（シャドウ・ダガー）
*   **闇市相場：** 金貨110枚
*   **効果：** 影を刺すと本体も動けなくなる。
*   **危険性：** 影が実体化して襲ってくる。自分の影に殺されると、自分が新たな影として地面を這いずることになる。

## 22. 悪夢の枕（ナイトメア・ピロー）
*   **闇市相場：** 銀貨20枚
*   **効果：** 楽しい夢が見られる（という触れ込み）。
*   **危険性：** 実際はサキュバス・イエス・ノー・マクラと呼ばれ、サキュバスに吸い尽くされることを了承する枕。精気を吸い尽くされ、二度と目覚めなくなる場合もあるが、当たりのサキュバスにあたることもある。

## 23. 透明化のマント（インビジブル・クローク）
*   **闇市相場：** 金貨400枚
*   **効果：** 姿を消せる。
*   **危険性：** 存在感まで消えすぎて、誰からも認識されなくなり、最終的に物理的に透過して地面に沈んでいく。

## 24. 時間停止の砂時計（クロノ・グラス）
*   **闇市相場：** 金貨1000枚～
*   **効果：** 時間を止める。
*   **危険性：** 自分も止まる。永遠に停止した時間の中で意識だけがあり続ける地獄を味わう。解除するには外部からの干渉が必要だが、時間は止まっているので誰も助けに来ない。

## 25. 鑑定虫眼鏡（アナライズ・レンズ）

*   **危険性：** 相手の「死期」も見えてしまう。知ってしまったストレスで胃に穴が開く。

## 26. 召喚の笛（パニック・ホイッスル）

*   **危険性：** 呼び出される魔物はランダムだが、なぜか「自分よりレベルが高い」魔物ばかり来る。呼び出した瞬間に捕食される。

## 27. 黄金の卵（ミミック・エッグ）

## 28. 契約の羽ペン（デビル・クイル）

## 29. 若返りの泉の水（リバース・ウォーター）

*   **危険性：** 分量の調整が難しく、一滴で10年若返る。うっかり一口飲むと受精卵まで戻って消滅する。

## 30. 幸福の青い鳥（ブルー・バード）
