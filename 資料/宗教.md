# 宗教・信仰勢力 設定案（たたき台）

王都および大陸における「宗教」は、精神的支柱であると同時に、**「回復・解呪・蘇生」というライフラインを独占する一大産業**としての側面を持つ。

## 1. 聖王教会（セイオウキョウカイ）

大陸最大勢力。王都に総本山を置く国教的存在。
「光の女神」を主神とするが、教義の核心は**「秩序と維持」**。

*   **社会的役割**：
    *   戸籍管理、結婚、葬儀の執行。
    *   **「回復魔法」「解呪」「アンデッド浄化」のライセンス管理**。
    *   ダンジョン内で死亡した冒険者の「蘇生儀式」を行えるのは、教会認定の司祭のみ（法的に独占）。
*   **腐敗と権力**：
    *   回復代金は高額。貧乏人は死ぬか、自然治癒に頼るしかない。
    *   「蘇生」の成功率は寄付金（上乗せ料金）で変わると噂されている。
    *   上位司祭は貴族並みの権力を持ち、ギルド『鉄人会』や『天耀の白鴉』とも癒着がある。

## 2. 狂天（キョウテン）周辺の信仰

ギルド『狂天』の構成員などが傾倒する、非公式・反社会的な信仰群。
統一された教義はないが、**「ダンジョン＝神、または神への通路」**と捉える傾向がある。

*   **深淵崇拝**：
    *   ダンジョンの深層に行くほど神に近づけると信じる。
    *   魔物を「天使」や「試練」と呼び、捕食されることを至福とする過激派もいる。
*   **禁術の実践**：
    *   教会が禁じる「生贄による強化」「死体改造」「魂の定着（リッチ化）」などを平然と行う。
    *   アカデミアを追放されたマッドサイエンティスト的研究者もここに混ざる。

## 3. ロプシーの信仰（精霊・土着信仰）

流浪の民ロプシーが持つ独自の自然崇拝。
教会からは「未開の迷信」、あるいは「異端」として冷遇されている。

*   **「契約」の概念**：
    *   神に祈るのではなく、土地の精霊や魔物と「契約」して力を借りるスタンス。
    *   これが彼らの「魔物使い」としての適性に繋がっている。
*   **教会との対立**：
    *   ロプシーの治癒術（薬草・呪術）は安価だが、教会法違反として度々弾圧される（教会の利益を侵害するため）。
    *   そのため、彼らは地下やスラムでこっそりと開業し、貧民の命綱となっている。

## 4. 宗教とダンジョンの関係（エコシステム）

*   **蘇生ビジネス**：
    *   冒険者が死体を持ち帰る → 教会が蘇生料を取る → ギルドが借金として肩代わりする → 冒険者はギルドに縛られる。
    *   この「死のサイクル」が王都の経済を回している。
*   **聖水・聖具の販売**：
    *   ダンジョン攻略に必須のポーションや聖水は、教会の認可工場で生産される。莫大な利権。
